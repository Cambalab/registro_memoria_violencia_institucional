# registro_memoria_violencia_institucional
Registro de historias sobre violencia institucional

Para descargar

```
git clone https://github.com/Cambalab/registro_memoria_violencia_institucional.git
```

Basado en Angular seed

Para instalar:

```
cd frontend
npm install
npm start
```
